Summary

  • This deck introduces you to the shiny app I made that shows US election results per state.
  • Red state - there are more votes to republicans than to democrats.
  • Blue state - there are more votes to democrates than to republicans.
  • White state - the votes are almost the same to both party candidates.
  • The darker the blue/ red color is, the bigger the differences are in votes.

Plot example

  • Results for 2016 (Clinton v.s. Trump). In the shiny app you can see results from 2012 as well for comparison
  • The app is here

Tech details

  • Plot was made with ggplot
  • Data was pre-processed and uploaded.
  • The function making plot is in helper.R to make the code neat.
